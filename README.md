Relevance Graph
=============
Put related nodes together by given gravitation between related nodes.


Dependencies
-----------

* kivy (for GUI, support Windows, Mac OS, Linux)


Run Demo
--------

6 node demo:

    $ kivy examples/demo_6_nodes.py

beers:

    $ kivy examples/beers.py
